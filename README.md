# Cancer Genomics Analysis

## Overview
This project analyzes somatic mutation data to identify:
- Frequently mutated genes
- Co-occurring mutations
- Mutually exclusive gene patterns
- Clinical correlations

## Data
- Mutation dataset (TSV format)
- Clinical data

## Methods
- Data cleaning using pandas
- Mutation matrix generation
- Co-occurrence analysis
- Mutual exclusivity detection
